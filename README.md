Allows to easily build-in encryption into your app, has Python wrapper and examples.
  * [Acra](https://github.com/cossacklabs/acra) ‚Äì Acra is a network encryption proxy to protect databases and database-based applications from data leaks, allowing greater security for distributed applications via cryptography and intrusion detection. Open source, has Python client libraries.
